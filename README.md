# Log Processor

Простой обработчик логов на Go. Читает CSV-файл, обрабатывает записи параллельно и выводит статистику.

## Что умеет

- Читает логи из CSV-файла
- Обрабатывает записи в нескольких горутинах (воркерах)
- Фильтрует ошибки (статус-коды 400-599)
- Считает статистику: всего запросов, ошибок, среднее время ответа
- Выводит результат в консоль
