# Log Processor

Обработчик логов на Go для анализа веб-серверных логов.

## Требования

- Go 1.21 или выше

## Формат входных данных

Программа ожидает CSV-файл со следующими полями (через запятую):
Timestamp,IP,Method,URL,StatusCode,ResponseTime
2024-01-15 10:30:00,192.168.1.100,GET,/api/users,200,150


| Поле | Тип | Описание |
|------|-----|----------|
| Timestamp | string | Дата и время запроса |
| IP | string | IP-адрес клиента |
| Method | string | HTTP-метод (GET, POST, etc.) |
| URL | string | Путь запроса |
| StatusCode | int | HTTP-статус код (200, 404, 500, etc.) |
| ResponseTime | int | Время ответа в миллисекундах |

## Что умеет

- Читает логи из CSV-файла
- Обрабатывает записи параллельно (5 воркеров)
- Считает статистику: всего запросов, ошибок (4xx/5xx), среднее время ответа
- Выводит топ IP-адресов по количеству запросов

## Структура проекта
Netology/
